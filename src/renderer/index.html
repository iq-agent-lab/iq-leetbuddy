PAT 발급 가이드

  1. 아래 링크를 클릭하면 scope가 미리 체크된 발급 페이지가 열림:
    → 새 PAT 발급하기 (repo scope, "iq-leetbuddy" description 미리 입력)
  2. Expiration: 본인 보안 정책에 맞춰. 'No expiration'도 OK (개인 도구 한정).
  3. Scopes: repo 하나만 체크 (이미 체크되어 있음). 다른 항목 다 무시.
    • private 레포까지 쓰려면 repo 통째
    • public 레포만 쓸 거면 public_repo 만 체크해도 OK
    • 그 외 (workflow, admin:org, delete_repo...) 전부 불필요
  4. 맨 아래 Generate token 클릭 → 한 번만 표시되는 토큰을 복사해 위 입력란에 붙여넣고 저장.

⚠️ 토큰은 한 번만 보여줘. 잃어버리면 재발급해야 함. 노출되면 즉시 revoke.
